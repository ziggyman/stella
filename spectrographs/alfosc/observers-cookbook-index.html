Observing Startup Procedure

Object Catalogue

Well in advance of performing the observations an object catalogue can be made. Note that it is also possible to enter targets on-the-fly and create a catalogue directly on the TCS while observing.

If you have submitted a catalogue by using our web interface NOT Source Catalogue Creation, you can load this catalogue on the TCS by the keypad keystrokes: Main, Catalogue, Load Catalogue and then typing /catalog/catname.cat

There is an online catalogue on the TCS which contains blank fields for twilight flats and focus stars. This one is loaded as above but typing only blanks. Instead of using the keypads you can type on the command line directly lo blanks.

Software Startup Procedure

To start up the instrument control software, see the Software Startup Procedure for the relevant instrument:
  • ALFOSC
  • StanCam
  • MOSCA
or see the NOTCam step-by-step guide.

RPC access code on TCS

Any instrument computer which needs access to the TCS, for instance to move the telescope during slit aquisition or dithering, will ask you about the current RPC access code at start-up. See BIAS start-up. This code is updated every day (around lunch time). When you are asked to supply the RPC access code, do as follows on the TCS terminal:
  • Go to Main Menu.
  • Press numerical key 8 "Status Pages".
  • Select numerical key 1 "Page number".
  • Then type 84 with the alpha-numeric keys to view page 84.
  • Read off the code as the first line on this page.

Obtaining Bias Frames

See the BIAS command list on how to obtain dark exposures of 0 seconds. For nicer FITS headers, set the image type, the object and the observer.

Acquire at least two bias frames in the beginning of the night and again in the end of the night. With two bias frames and with two flat fields of the same level, gain and readout noise can be determined, e.g. by using the IRAF task findgain.

Telescope Control System (TCS)

  1. Turn up the brightness of the TCS screen. (on right-hand side, below the screen)
  2. Look at the top-left of the TCS screen. If item ST is incrementing, then the TCS is running, if not - then the TCS needs resetting (see the TCS Manual - in a binder in the control room) .
    Type refresh if the screen looks strange .

Telescope power-on and Dome Opening

  1. Check safety system in the computer room (See TCS Manual Sect. 10.1 for procedure) by pressing first the left and then the right reset button by the red LED lamps and then turning the left and the right keys. If you get a repeated clunking sound then the system has not reset, check the red lights above to see if a safety stop or other item is set.
  2. Type power-on at TCS and confirm with y for YES. Wait for the message Telescope Power is ON before continuing. This takes about 1 minute.
  3. At sunset, if weather conditions are OK (see wind/humidity/dust restrictions on the wall in the control room), open upper hatch with mirror covers closed.
    Use keypad keystrokes: Main, Start/Stop, Open Upper Hatch. This takes about 3.5 minutes.
  4. In dome (While upper hatch opens):
    • open side ports (again, only if conditions are OK ). Do not use too much force, but use the handle for lifting the ports if needed -you could damage the side ports!!.
    • check that nothing is in the way of telescope motion (6.5-90 degs altitude)
  5. Open lower hatch (again, only if weather conditions are OK. In case you can open the upper hatch but not the lower hatch then the maximum zenith distance for observing is 55 degrees ). Use keypad keystrokes: Main, Start/Stop, Open Lower Hatch. This takes about 2 minutes.
    Do not open lower hatch before upper hatch has completed opening.
  6. While lower hatch is opening, mirror covers can be opened as well. Use keypad keystrokes: Main, Start/Stop, Open Mirror Covers. This takes about 50 sec.
    Please note that power for hatches and mirror covers is completely independent of the telescope power and electronics in Control room.

Field Rotator

At the TCS terminal give the rotator setup commands as indicated in the table below:

Instrument in use TCS Command Orientation of images
ALFOSC rot-auto
field +90		 North up and East to the left.
Slits are then oriented E-W (P.A. = +-90).
TurPol or StanCam rot-auto
field 0		 NB! North up and East to the right
NOTCam or MOSCA rot-auto
field -90		 North up and East to the left (MOSCA).
NB! East up and North to the left (NOTCam).

Note that if you do spectroscopy only, setting the field will usually not be useful as you probably would like to observe on the parallactic angle, in which case the field will be different for each target. To observe in this mode, simply do:

  1. Define-Parallactic-Instrument-Angle XX or just d-p XX at the start of your run (sticky command).

    d-p 0 ALFOSC
    d-p -90 SOFIN and NOTCam

  2. For presetting the telescope use Catalogue function 7 "Preset w par ang" instead of the usual 4 "Preset to this".
Upon every preset the rotator position will be close to the value defined as above at start. Note that the rotator should be set to automatic mode (command rot-auto on the TCS) in order to keep a constant field orientation, such that it will gradually move away from zero. If you want to know your current field angle, for instance to be able to compare an aquisition image with a finding chart, then type on the TCS field and then press the HELP key.

We refer to the different instrument pages for specific info on spectroscopy with: ALFOSC, SOFIN, and NOTCam.

CCD Probe

The CCD probe is used for the standby Camera (StanCam) only.
  1. To use the StanCam, put the CCD probe in the CCD position with c-p-c.
  2. When not using StanCam, put the CCD probe in the park position with c-p-p.
For the use of StanCam we refer to documentation on The StanCam Page.

Calibration frames

The type of calibration frames needed depend very much on your science goals and the instrument mode you are using. We recommend our documentation on the different instrument pages. Please, check the User Guides available for: ALFOSC, Stancam, NOTCam, MOSCA, Turpol, and SOFIN. Below is a short guide regarding bias frames and twilight flat fields for optical imaging.

Flat field frames:

  1. Remember to set tv-off before making sky flatfields.
  2. Load catalogue (Main, Catalogue) with empty fields by using the command lo blanks
  3. Select an empty field with the keypads 5,6,8,9 and preset the telescope with the keypad 4
  4. With BIAS (see instrument user manuals for ALFOSC, Stancam, and MOSCA, make flat fields in the desired filters. It is recommended to:
    - rotate the field 90 degrees for each image. Use the commands field 0, field 90,, etc.
    - make small offsets between the images, in order to be able to get rid of stars (on the TCS set offset size with the command o-s 10 10 and then use arrow keys to make the actual offsets while the CCD is reading out). It is recommended to be close to focus for each filter, and to use an offset size at least 10 times the size of the FWHM of the seeing disk in order to easily filter out the stars.

Guide Probe

  1. Switch on the two Guide TV monitors above the TCS monitor.
  2. When it is dark outside, type tv-on to power up the TV guide camera.
    Note: tv-off switches the TV camera off.
    Remember to set tv-off before taking dawn Sky flatfields.

Hereafter the following is made automatically:

  • TV-Focus follows the focal plane curvature when the guideprobe is moved in X and Y.
  • When presetting to an object, the guideprobe moves to a suitable guidestar (See the section Autoguiding) and chooses an "appropriate" TV filter based on its brightness. (This might need adjustment in the case of bad seeing or clouds.)

Focusing the telescope

The telescope focus depends on temperature and zenith distance. These relations have been measured to a high accuracy and the corresponding corrections are applied in real time by the TCS. (Note that you will not see these corrections in the focus value displayed on the TCS. To avoid confusing the observer this one is kept constant.) The focus should therefore remain stable during not only a night but a complete run. We do, however, encourage observers to check the focus a couple of times during the night. Below we describe how to set and monitor the focus for our different instruments.
  1. Find a nearby field with focus stars in the blanks catalogue ( lo blanks).
  2. Select focus field with the keypads 5,6,8,9 and preset telescope with the keypad 4
  3. With the image acquisition software BIAS (see instrument user manuals), make focus sequences and determine the best focus value or -offset
  4. Determine focus offsets for the filters to be used, or use the determined filter offsets.
  5. In the TCS, enter the new focus with the command foc-pos or the focus offset with the command foc-del
  6. To focus the Guide TV, first acquire a star on the TV:
    • If no star is seen in the Guide TV, open Guide TV filter with the command tv-fil o. Other options instead of o are: c (closed), b (blue), y (yellow), r (red) and g (gray, a 5 mag neutral density filter.)
    • Is there still no star, move the Guide probe to center position with the command g-p-c to view the focus star itself.
  7. When a star is visible, focus the Guide TV with the keypad keystrokes: Main, Guideprobe, Others, Tv Focus+/- , or command tv-foc nnn
  8. Move the Guide Probe to park position with the command g-p-p

Using ALFOSC

When focusing ALFOSC we use the 'focus pyramid', an optical device which splits the light beam into four segments, each of them producing an image of the star on the CCD. The distance between the four images is linearly dependent on the focus of the telescope. By measuring the distance between the four images of a star we directly get the correct focus of the telescope - independently of seeing variations.

Follow the ALFOSC focus procedure.

Filter name NOT filter number foc-del
Bessel U #7 -70
Bessel B #74 -85
Bessel V #75 -10
Bessel R #76 -80
Interference i #12 -5

The focus offsets (with respect to "no filter") for the standard UBVRi broadband filters used in the ALFOSC filter wheel are listed in the table to the right. For more filters, check the determined filter focus offsets.

When you have corrected the focus, it is appropriate to check it 3 times. The focus value you have found for "no filter" in the beam is the right focus to use for ALFOSC spectroscopic mode.

Note that the internal ALFOSC focus has default value 1877 and is found to be very stable. No need to change it (unless you use the spectro-polarimetry mode).

For ALFOSC polarimetry the recommended focusing procedure is a bit different from above (see About focusing FAPOL). For spectro-polarimetry the internal ALFOSC focus must be changed (to 900) in order to have the polarimetric slitlet in focus. Compared to normal ALFOSC use a substantial focus offset is required by putting in the retarder plate (either the half-wave or the quarter-wave, which have different thickness and therefore different focus shifts) and the calcite plates in the focal plane (aperture wheel).

Using StanCam

A traditional focus sequence should be made exposing at different focus values and stepping the telescope such that a number of focus measurements can be read out on the same image. This is done in StanCam BIAS with the command:

focus [focus startvalue] [focus stepsize] [ttt]

where you should enter a focus startvalue close to the default focus for StanCam (23080), a focus stepsize of around 50 during the first iteration, and an exposure time preferrably as long as 15 seconds to filter out possible rapid seeing variations.

When the command is finished and the image is read out you will find your focus star stepped along the CCD, the last measurement being offset 90 degrees with respect to the others. Estimate the sharpness and roundness of the exposures with imexam and possibly iterate with a smaller focus stepsize around the best value.

Using MOSCA

The same procedure as for StanCam, but default focus for MOSCA is 23250.

Using NOTCam

Default telescope focus for NOTCam is 23500 for the K bands in imaging mode with the WF camera (when the internal camera focus is set to 6000). Check the NOTCam User's Guide under section Focusing for instructions regarding imaging mode focusing.

Check NOTCam spectroscopy for focusing instructions in this mode.

Autoguiding

It is advantageous to have the telescope `autoguide' on an optical source close to your target object. By autoguiding we mean let the telescope control software (TCS) monitor the position of a reference star and correct the 'pointing' of the telescope quickly to correct for drifts in tracking. Autoguiding will produce the best image quality - especially away from zenith.

At the NOT we have implemented a system whereby whenever the telescope is positioned preset to an object, the TCS automatically selects a suitable star from the HST guide star catalogue. The guide probe is then automatically placed at the correct position to auto-guide on this star. This saves considerable (and valuable) observing time over manual searches for guide stars. When the telescope instrument rotator (field angle) is manually changed (e.g. for slit rotation) the TCS will automatically re-select a guide star for the new field angle. This means that retyping your current field angle on the TCS is a way to research for a guide star at current position (if you somehow lost it).

Once a guide star has been selected the observer simply starts the autoguider. The TCS always selects the best (brightest) guide star available at the current position and displays it on the Guide Monitor. To start autoguiding do the following:

  1. If the guide star looks very faint or you can't see it, check that TV filter is open or type on the TCS tv-fil o.
  2. If still too faint then start TV integration with the command int-fr n where n is number of frames to integrate, say 2 - 5. A small window pops up with the integrated image. Place the Autoguider mouse pointer on the star and click the right button labelled INT. Follow the steps below as for normal autoguiding, but grab the star in that small window. To stop integration mode, type int-fr 0. Note that the guide star should not be on the right edge of the screen because the INT box will not reach there.
  3. Place the pointer accurately on the star by moving the Autoguider mouse.
  4. Click the middle button labelled STAR to put the box onto the star.
  5. Enter the command a-g to start autoguiding without relaxation, or rel-a to start autoguiding with relaxation (the box is then moved to the star without moving the telescope).
  6. On the TCS monitor the signal strength from the autoguider is shown as a number. Anything between 1 and 100 is useful. If your signal is above 100 then put in a filter (red, yellow, or grey) tv-fil r, tv-fil y, or tv-fil g. If your signal is < 1 then guiding will be lost and you should use integration mode.
  7. To stop autoguiding again, use the command a-s
Note: make sure the small box is outside the star box -if needed, move it by clicking the button SKY (can be moved only while not autoguiding). This is the sky measurement box. You can change the star box size depending on the seeing by typing b-s xx, where xx is a number usually around 18-20.

If you see no guide star, then:

  1. Check that TV filter is open (tv-fil o).
  2. Check guide camera focus (set it by the command tv-foc xxx).
  3. Check the VT100 terminal to your upper left which displays the magnitude of the latest acquired guide star. Depending on the seeing and possible clouds it could be that the best guide star in this field is too faint. If so, consider whether you could use another field angle.

If you lose the guide star after a telescope offset, then:

Give TCS command field xx.x, where xx.x is the current field angle. This leads to no movement of the telescope, but a guidestar is re-acquired.

Closing Down Procedure

  1. BEFORE YOU MAKE FLATFIELDS, turn Guide TV off with command tv-off, then take your flats.
  2. PUT THE TELESCOPE TO ZENITH. Use keypad sequence : MAIN, Start/Stop, Zenith Position. This turns off the GUIDE TV and parks the GUIDE probe. Wait for "Rotator -90".
  3. PARK THE BUILDING. Use keypad sequence : MAIN, Start/Stop, Park Building. This process can take up to 8 min. Building will stop at Az = 116, Alt = 90.
  4. CLOSE THE MIRROR COVERS. Use keypad sequence : MAIN, Start/Stop, Close Mirror Covers. This takes 42 sec.
  5. CLOSE THE LOWER HATCH. Use keypad sequence : MAIN, Start/Stop, Close Lower Hatch. This takes about 3 min.
    If timeout occurs see TCS Manual p. 43.
  6. CLOSE THE UPPER HATCH. Use keypad sequence : MAIN, Start/Stop, Close Upper Hatch. This takes about 5 min.
    If timeout occurs or hatch does not move, see TCS Manual p. 43.
  7. CLOSE THE SIDE-PORTS. Go into dome and close the 4 side-ports. Please, do not use violence when closing the side-ports. If stuck, use the handle to lift the port, then close.
  8. POWER DOWN THE TELESCOPE. After all of the above have been completed, on TCS type power-off and wait until message Telescope power is off appears.
  9. TURN DOWN THE TCS SCREEN BRIGHTNESS. Use the knob on the right-hand side of the TCS monitor, near the power switch.
  10. TURN OFF MONITORS. Turn off the two autoguider monitors and dome TV camera monitor.
  11. HAVE A LOOK IN THE DOME at hatches and telescope, to check for any faults that might have occurred.
  12. COMPLETE THE WWW REPORT FORMS. Fill in and submit Internet NOT End-of-Night Report. Be sure to include any faults that occurred and submit a fault report, so we can fix them promptly!
    If it is the last night of the observing run, also fill in and submit Internet NOT End-of-Run Report.
  13. LOCKUP THE DOME. Lock the outer dome door when you leave the building
  14. CLEAN UP THE KITCHEN/LOUNGE AREA!. In the service building, please wash up all dishes and cups you used during the night. Switch off all lights when you leave.
  15. LOCKUP THE SERVICE BUILDING. Lock the NOT service building outer door when you leave for the Residencia.

-Thank you for your co-operation.

Tracking of moving targets

The NOT was not originally built for tracking of moving targets (planets, comets, asteroids, satellites), but there are currently two ways of doing this, depending on the speed of the object.

Slowly moving targets

For objects moving rather slowly, i.e. less than about 1 arcmin during an exposure, the tracking is handled by moving the guide box on the autoguider screen, while autoguiding, in order to compensate for the object motion.

  1. Calculate your object's motion in RA and DEC rates in advance. NB! Rates are in arcseconds per hour!

  2. Preset to your object, and get a guide star (hopefully) on the guide monitor.

  3. Position the box in an appropriate place on the screen. Then move the guide probe to put the guide star in the box: first set the step size by p-s-s <1000> <1000>, then go to MAIN MENU - GUIDE PROBE - OTHERS, and use the X+,Y+,X-,Y- keys to move the probe. When the star is in the box, start autoguiding with the TCS command a-g.

  4. Start then tracking of the moving object by the TCS command Box-Motion-Start <RA-rate> <DEC-rate> or in short ag-bo-on <RA-rate> <DEC-rate>. If the box reaches the opposite edge of the monitor, it stops moving. You can try with a short box move with much higher rate, to find out in which direction the box moves.

  5. Take the exposure, and then stop tracking by Box-Motion-Stop or in short ag-bo-of.

Rapidly moving targets

For more rapidly moving targets, use the set-rate command to continuously move the telescope from a pre-calculated change in position. In this mode autoguiding is not used!

  1. Preset to your object.

  2. Set the rate of change by set-rate <RA-rate> <DEC-rate> (NB! Rate in arcsec per second!). The maximum rate is 1200 arcsec/s.

  3. Make the exposure.

  4. Reset by set-rate 0 0 .

For experts only

If the target has an unknown or rapidly changing motion, you might want to adjust the rate of change in RA and DEC while tracking. Then, use the telescope in set-mode, as opposed to the default offset-mode. In set-mode the cursor keys (which in offset-mode are used to move the telescope with a given offset step), are now used to adjust the rate of change in RA and DEC, once set with set-rate. As in the above case, autoguiding is not used.

The step of the adjustment is set with the command set-step, which can take values from 0.0009 to 1200.0 arc seconds per second (NB! Note the units!). The Up/Down keys adjusts the DEC set rate with +/- the set-step size, and the Left/Right keys adjust RA set rate with -/+ the set-step size.

  1. Preset to your object.

  2. Type in the TCS command set-mode

  3. Guess the starting set-rate <RA rate> <DEC rate> values.

  4. Set the size of adjustment by set-step <size> and use the cursor keys to adjust the speed.

  5. Go back to offset-mode when finished tracking this object, or before presetting to another object.

NB! This mode is quite tricky to use correctly, and needs some practicing.

Before calling NOT staff

If you have ...
  • ... no light on the CCD -- check:
    • Dome open, mirror cover open
    • Lower hatch not obstructing at low altitude
    • Guide probe out of light path
    • CCD (StanCam) probe in park position
    • FASU shutter open
    • Correct filter in instrument
    • Correct telescope focus
      • Check all the above again. If still no joy, then phone duty staff.
  • ... wrong field on CCD -- check:
    • Coordinates of field
    • Epoch of coordinates
    • No typos in entering coordinates in TCS
    • Field rotation is correct
  • ... no guide star on guide probe screen -- check:
    • Dome open, mirror cover open
    • Guide probe in correct position
      (check field and rotator angle)
    • TV switched on
    • TV filter open
    • TV focus correct
  • ... lost tracking/guiding:
    • How close to the zenith are you?
      (Good guiding and tracking is guaranteed only 5 degrees or more from zenith).

If you enocunter a fault or a problem with instrument, computers or telescope during the night, then always fill in a Fault Report.
