WiFeS IRAF package

PLEASE NOTE: The WiFeS IRAF package is NOT being officially developed, updated or supported by RSAA. There are a number of users who are using it and have been making their own improvements. Some of these users can be contacted by emailing the WiFeS users mailing list: wifes-users@mso.anu.edu.au, which is used for sharing information and problems regarding WiFeS use, data and data reduction. Please send requests to join or leave this mailing list to sysman@mso.anu.edu.au. There is an email archive for the WiFeS users' list here.

The WiFeS data reduction package is an external IRAF package, which is based on and utilises the Gemini Observatoryâ€™s Gemini IRAF package.

To use the WiFeS package, then, you need a recent IRAF installation (the package has been tested with version 2.14.1), and the Gemini package (wifes has been tested with versions 1.9 and 1.10), which has its own prerequisites, as described on the Gemini Observatory web pages. Note that unlike the latest version of the Gemini package, the WiFeS package is not PyRAF-friendly.

A version of the WiFeS IRAF package is available here: WiFeS package files The file wifes.tar.gz (~360MB; updated Feb 2011) contains the IRAF package scripts and data files that are installed in the same way as a typical external IRAF package. The README file in this package (and found on the ftp page) has instructions for installing and updating the scripts.

PLEASE NOTE the following known limitations:

  • The atmospheric refraction correction routines in this version of the package do not work on some systems (e.g. Mac).
  • The scripts have not been fully tested with data obtained with the 'stellar'/'half-frame' readout mode, or the fast readout mode for the red camera where the full frame is readout using only two amplifiers. Check the results and output carefully when reducing data in these formats.

A draft data reduction manual exists, but it does not incorporate recent changes in the version of the package available above. In particular, it does not include descriptions of the several methods that have been added to attempt to remove the bias level and shape from WiFeS data and care must be taken to make sure that However, it describes how to install the WiFeS IRAF package and contains basic information about the processing tasks and how to use the WiFeS scripts: Draft user's manual.

An example set of (very old) calibration files (e.g. for quick-look reduction) is also provided as a separate tar file for download (~1GB): wifes_cal.tar.gz. When extracted, the move the calibration files into the '/cal' directory inside the top 'wifes/' directory of the package tree

Wavelength calibration data

The available arc lamp line lists are found in the data/ directory of the WiFeS IRAF package. References solutions are provided in package (cal/ directory inside the package) for some combinations of arc lamp and instrument configuration (grating and dichroic settings). There are also a number of reference data provided in cal/linelists, including plots of spectra with labelled arc lines. Some examples are provided below.

CuAr arc:

  • R3000/RT560: File:R3000 CuAr spectrum.pdf
  • B3000/RT560: File:B3000 CuAr spectrum.pdf

The NeAr lamp in the WiFeS calibration unit was replaced in August 2009 with a new lamp that now contains some Fe emission lines in addition to Ne and Ar. Here is a line list kindly identified and provided by Mike Bessell: File:FeArNe.txt. The data reduction package includes reference solutions for this lamp with the standard combinations of grating and dichroic (as of 31-10-2009).

(Fe)NeAr arc:

  • U7000/RT480: File:U7000 FeNeAr spectrum.pdf
  • B7000/RT615: File:B7000 FeNeAr spectrum.pdf
  • R7000/RT480: File:R7000 FeNeAr spectrum.pdf
  • I7000/RT615: File:I7000 FeNeAr spectrum.pdf


Fe-Ar Line List: File:WiFeS-Fe-Ar-lines.txt

Sky spectrum, R3000 grating: File:R3000 sky spectrum.pdf; sky line list: File:R3000 sky linelist.txt

OH night sky lines List: File:WiFeS-OH-lines.txt


In the first instance, refer to the WiFeS observer's manual, which is available for download as a .pdf. For any other relevant comments you would like to be included, please add them here. (Mike Dopita)
