estimatescatter


Usage:

estimatescatter "char[] [@]FitsFileName_In" [AREA=[,,,]]

Parameters:

char[] [@]FitsFileName_In

Name of input fits image (without @, e.g. "image_in.fits"), or name of input file list (with @, e.g. "@images_in.list")

char[] [@]FitsFileName_Out

Name of output fits image (without @, e.g. "image_out.fits"), or name of output file list (with @, e.g. "@images_out.list")
* output file list must have same number of elements as input file list

int ClusterSizeX

size of rectangular CCD areas in x to calculate scattered light from

int ClusterSizeY

size of rectangular CCD areas in y to calculate scattered light from

optinal parameter: AREA=[,,,]

CCD area of interest (pixel values range from 0 to size-1)
