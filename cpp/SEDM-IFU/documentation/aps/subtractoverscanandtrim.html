subtractoverscanandtrim


Usage:

subtractoverscanandtrim "" "" <[int(xmin),int(xmax),int(ymin),int(ymax)] OverscanArea> <[int(xmin),int(xmax),int(ymin),int(ymax)] TrimArea>

Parameters:

char[] [@]FitsFileName_In

Name of input fits image (without @, e.g. "image_in.fits"), or name of input file list (with @, e.g. "@images_in.list")

char[] [@]FitsFileName_Out

Name of output fits image (without @, e.g. "image_out.fits"), or name of output file list (with @, e.g. "@images_out.list")
* output file list must have same number of elements as input file list

[int(xmin),int(xmax),int(ymin),int(ymax)] OverscanArea

CCD area containing the overscan (pixel values range from 0 to size-1)

[int(xmin),int(xmax),int(ymin),int(ymax)] TrimArea

CCD area containing good data (pixel values range from 0 to size-1)
