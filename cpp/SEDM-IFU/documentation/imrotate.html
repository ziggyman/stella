imrotate


Usage:

imrotate "char[] (@)FitsFileName_In" "char[] (@)FitsFileName_Out"

Parameters:

char[] [@]FitsFileName_In

Name of input fits image (without @, e.g. "image_in.fits"), or name of input file list (with @, e.g. "@images_in.list")

int I_AngleClockwise

Clockwise angle to rotate input image(s) [90,180,270]

char[] [@]FitsFileName_Out

Name of output fits image (without @, e.g. "image_out.fits"), or name of output file list (with @, e.g. "@images_out.list")
* output file list must have same number of elements as input file list
